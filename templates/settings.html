How does it work?

TP (Take Profit) and SL (Stop Loss) levels are automatically calculated based on the ATR (Average True Range) indicator:

  • ATR Period: ATR is calculated based on the volatility of the last X bars.
  • TP Multiplier: TP price is set by adding (long) or subtracting (short) ATR x TP Multiplier to the entry price.
  • SL Multiplier: SL price is set by subtracting (long) or adding (short) ATR x SL Multiplier from the entry price.
Number of bars to use for ATR calculation (e.g. 14)
Multiplier for Take Profit (e.g. 2.5)
Multiplier for Stop Loss (e.g. 3.0)
Example Calculation

Entry Price: $97,000, ATR: $500, TP Multiplier: 2.5, SL Multiplier: 3.0

LONG Position:
  • TP: $97,000 + (2.5 x $500) = $98,250
  • SL: $97,000 - (3.0 x $500) = $95,500
SHORT Position:
  • TP: $97,000 - (2.5 x $500) = $95,750
  • SL: $97,000 + (3.0 x $500) = $98,500
